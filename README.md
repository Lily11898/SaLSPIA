# SaLSPIA: Spectrally Adaptive LSPIA for B-Spline Fitting

MATLAB reference implementation accompanying the paper:

> Spectral Adaptivity for Iterative Least-Squares B-Spline Curve and Surface Fitting  
> Xingxuan Peng and Yutong Li

The repository contains the proposed SaLSPIA implementation, the comparison
methods used in the manuscript, experiment drivers, and redistributable input data.
See `DATA_SOURCES.md` before publishing or redistributing third-party data.

## Requirements

- MATLAB R2021b.
- No external toolboxes are required.
- Scripts that generate random data use a fixed `rng(...)` seed.

## Quick Start

```matlab
cd SaLSPIA_release
setup_paths
cd experiments
run_surface_comparison
```

`setup_paths` adds `algorithms/`, `utils/`, and `experiments/` to the MATLAB path.

## Layout

```text
algorithms/   SaLSPIA, LSPIA, ALSPIA, MLSPIA, and ablation routines
utils/        B-spline collocation and curve parameterization helpers
experiments/  Drivers for the manuscript experiments
data/         Redistributable data and local-data placeholders
results/      Generated outputs from selected drivers
```

## Experiment Conventions

- Curves use chord-length parameters and the least-squares knot construction in
  `utils/make_params_knots.m`.
- Tensor-product surfaces use normalized uniform parameters and uniformly spaced
  clamped knot vectors in each parametric direction.
- The default SaLSPIA parameters are `sigma = 1e-4`, `M = 10`, and
  `delta = 1e-8`.
- The noisy-curve driver uses the fixed sequential random stream
  `rng(202608, 'twister')`, generating blob noise before helix noise.

## Data

The analytic blob, helix, Dini, and Peaks examples are generated inside the
experiment scripts. The LiDAR terrain files bundled in `data/` are derived from
the LINZ Auckland LiDAR 1m DEM and are distributed with attribution under
CC BY 4.0.

The reindeer contour (`cur_data deer`) and G-loop curve
(`s_loop_curve_data.txt`) are required by some experiments but are not bundled in
the public release unless redistribution permission has been confirmed. See
`DATA_SOURCES.md` for source links and installation paths.
